# Onboarding

1. Clone Repository und installiere Abhängigkeiten für Backend und WebUI.
2. Starte Postgres + Flowise via `flowise/scripts/start-flowise.sh`.
3. Setze `.env` im Backend (siehe `env.ts` Defaults) und starte `npm run dev`.
4. WebUI via `npm run dev -- --host --port 4177`.
5. Nutze die Onboarding-Checkliste im Frontend, um alle Systeme zu validieren.
